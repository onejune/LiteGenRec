# LiteGenRec 自回归生成实验总结

## 实验结果

| 方案 | AUC | vs DeepFM | vs HSTU-Lite V3 |
|------|-----|-----------|-----------------|
| V1 自回归生成 | 0.7235 | -2.37bp | -6.18bp ❌ |
| V2 掩码特征预测 | 0.7571 | +0.99bp | -2.82bp ⚠️ |
| DeepFM 基线 | 0.7472 | - | -3.81bp |
| HSTU-Lite V3 | 0.7853 | +3.81bp | - |

## 结论

1. **生成式建模在 CTR 场景效果不佳**
   - CTR 预测本质是判别式任务
   - 自回归生成无法学习有效模式

2. **掩码预训练有正收益**
   - 优于 DeepFM +1bp
   - 但不如直接特征交互

3. **判别式架构仍是主流**
   - HSTU-Lite V3 (0.7853) 仍是最优
   - 特征交互比特征重建更重要

## 下一步方向

1. **回归判别式架构**
   - 优化特征交叉 (FM/DCN)
   - 多任务学习 (CTR+CVR)

2. **放弃生成式路线**
   - CTR 场景不适合生成式建模
   - 推荐场景的生成式更适合物品 ID 预测

3. **探索其他方向**
   - 模型压缩与部署优化
   - 少样本学习
   - 冷启动场景
