# Exp01: SID Construction for Ads

## 目标

探索广告场景下的 Semantic ID 构建方案，将广告编码为语义化的离散 token。

## 背景

- Semantic ID (SID) 是生成式推荐的基础
- 好的 SID 应该：语义相似的广告 ID 相近，碰撞率低
- 广告比商品复杂：包含创意、文案、商品、广告主等多维信息

## 方法

| 版本 | 方案 | 描述 |
|------|------|------|
| v1 | Text Only | 仅用广告标题+描述 → Embedding → RQ-VAE |
| v2 | Text + Category | 加入类目信息 |
| v3 | Multi-modal | 文本 + 图片特征 |

## 评估指标

- **碰撞率**: 相同 SID 的广告比例 (越低越好)
- **语义分组准确率**: embedding 最近邻中同组 SID 比例 (越高越好)
- **重建误差**: RQ-VAE 重建 loss
