# Exp06: Semantic ID 实验

## 实验目标

使用 VQ-VAE 生成层次化语义 ID，探索语义化特征表示对 CTR 预测的影响。

## 方法论

基于 TIGER (NeurIPS 2023) 的思路：
1. 将特征嵌入通过 VQ-VAE 编码为多层语义 ID
2. 使用语义 ID 作为新的特征表示
3. 层次化 ID 捕获不同粒度的语义信息

## 实验版本

| 版本 | 方法 | 结果 |
|------|------|------|
| V1 | VQ-VAE + 语义ID | AUC 0.5000 ❌ 失败 |
| V2 | 简化版 | - |
| V3 | 改进版 | AUC 0.7492 |

## 失败原因分析 (V1)

1. **向量量化维度问题**: VectorQuantizer 输出 indices 形状错误
2. **梯度断裂**: VQ-VAE 的离散化导致梯度传播困难
3. **语义 ID 质量**: 层次化编码未学到有意义的语义

## 改进措施 (V3)

- 简化量化器结构
- 调整 codebook 大小
- 使用软量化替代硬量化

## 实验结果

| 模型 | AUC | 备注 |
|------|-----|------|
| VQ-VAE V1 | 0.5000 | 随机猜测级别，失败 |
| V3 改进版 | 0.7492 | 低于基线 |

## 关键发现

❌ **VQ-VAE 在 CTR 场景不适用**
- 特征本身已经是高基数稀疏特征，再量化会丢失信息
- 层次化语义 ID 对推荐场景不一定有效 (与论文场景不同)

## 结论

Semantic ID 方法更适合生成式检索场景，不适合直接的 CTR 预测任务。
