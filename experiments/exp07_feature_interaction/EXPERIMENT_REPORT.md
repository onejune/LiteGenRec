# Exp07: Feature Interaction 实验

## 实验目标

结合 DCNv2 和 AutoInt 的特征交叉能力，提升模型性能。

## 模型架构

```
Input: 26个类别特征
  ↓
Embedding Layer
  ↓
┌─────────────────┬─────────────────┐
│   DCNv2 路径    │   AutoInt 路径   │
│  交叉网络 ×3层   │  自注意力 ×2层   │
└────────┬────────┴────────┬────────┘
         ↓                 ↓
         └───── Concat ─────┘
                 ↓
            MLP Head
                 ↓
            CTR 预测
```

## 实验配置

- **DCNv2**: cross_layers=3
- **AutoInt**: num_heads=4, num_layers=2
- **embed_dim**: 32
- **epochs**: 3

## 实验结果

| 指标 | 值 |
|------|-----|
| AUC | 0.7532 |
| 参数量 | ~4.1M |

## 与基线对比

| 模型 | AUC | 备注 |
|------|-----|------|
| DeepFM 基线 | 0.7472 | +0.60% |
| SimpleGenCTR V1 | 0.7663 | -1.31% ❌ |
| **DCNv2+AutoInt** | 0.7532 | 本实验 |

## 分析

1. **优于基线**: 相比 DeepFM 提升 0.6%
2. **不如 Transformer**: 简单的特征交叉不如自适应注意力
3. **模型复杂度高**: 参数量翻倍，收益有限

## 关键发现

- 显式特征交叉 (DCNv2) + 自注意力特征交互 (AutoInt) 有一定效果
- 但不如纯 Transformer 的端到端学习效果好
- 模型复杂度增加带来的边际收益递减

## 结论

⚠️ 特征交叉组合方法有效但非最优，Transformer 端到端学习更优
